Lenguajes de marcas¿Qué son los lenguajes de marcas? Los lenguajes de marcas o Markup Language son maneras de codificar documentos que incorporan etiquetas, marcas o anotaciones con información adicional sobre la estructura del texto o su forma de presentación. Mediante los lenguajes de marcas, con la utilización de las etiquetas, se puede hacer explícita información del documento, como por ejemplo su estructura, su contenido semántico o de cualquier otro tipo. ¿Ejemplos de lenguajes de marcas? Entre los ejemplos de lenguajes de marcas están los siguientes: RTF (Rich Text Format - Formato de Texto Enriquecido): Es un lenguaje de representación desarrollado por Microsoft en 1987. Representa documentos de texto con formato de presentación añadido. TeX: Es un lenguaje de representación que representa fórmulas matemáticas complejas. eXtensible Markup language (XML)Definición de XML Como definimos en el apartado anterior, XML es un lenguaje descriptivo, es decir, por definición se centra en el contenido, estructurando el documento en partes, pero omite la representación. XML (eXtensible Markup Language) es un meta-lenguaje que permite definir lenguajes de marcas desarrollado por el World Wide Web Consortium (W3C) utilizado para almacenar datos en forma legible. Proviene del lenguaje SGML y permite definir la gramática de lenguajes específicos para estructurar documentos grandes.
